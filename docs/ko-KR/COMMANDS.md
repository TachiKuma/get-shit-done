## 문서 위치

- 이 문서는 summary 전용 페이지이며 `commands/gsd/*.md`의 전체 본문 mirror 가 아닙니다.
- Phase 04 범위에서는 핵심 워크플로우 명령만 간단히 안내합니다.
- 고급 플래그, 주변 명령, 최신 세부사항은 [English canonical](../COMMANDS.md)로 fallback 합니다.

## 핵심 명령 요약

| Command | 목적 | Fallback |
|---------|------|----------|
| `/gsd-progress` | 현재 진행 상태와 다음 권장 작업을 빠르게 확인합니다 | 자세한 내용은 [English canonical](../COMMANDS.md) |
| `/gsd-discuss-phase [N]` | planning 전에 구현 결정과 범위를 정리합니다 | 자세한 내용은 [English canonical](../COMMANDS.md) |
| `/gsd-plan-phase [N]` | research, planning, verification 흐름을 실행합니다 | 자세한 내용은 [English canonical](../COMMANDS.md) |
| `/gsd-execute-phase <N>` | phase 계획을 실행하고 SUMMARY 및 검증 단계로 넘깁니다 | 자세한 내용은 [English canonical](../COMMANDS.md) |
| `/gsd-next` | 현재 상태를 읽어 다음 논리적 단계를 자동 선택합니다 | 자세한 내용은 [English canonical](../COMMANDS.md) |

## Fallback 규칙

- 이 페이지는 summary only 입니다.
- 전체 syntax, flags, options, examples 는 항상 [English canonical](../COMMANDS.md)을 사용하세요.
